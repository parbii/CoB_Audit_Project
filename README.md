# PVAMU College of Business (CoB) Communication Audit & Centralization

## Overview
Led a strategic audit and data migration project to centralize communication for the Prairie View A&M University College of Business (CoB). The goal was to eliminate information silos and establish a professional Microsoft Teams ecosystem for 500+ students and organization leaders.

## Technical Implementation: Web Scraping & Data Extraction
To ensure the audit was based on real-time data, I developed a custom **Python** script to scrape organization data from **PawLink**.
* **Automation:** Used **Playwright** and **BeautifulSoup** to programmatically extract organization names, primary contact emails, and physical office locations.
* **Data Cleansing:** Processed the raw HTML data to identify outdated contact information and redundant listings.
* **Verification:** Validated scraped data against the Dean’s current records to ensure 100% accuracy before migration.

## The Strategy Group Workflow
* **Stakeholder Interviews:** Engaged with the Dean’s Student Advisory Council to map current communication gaps.
* **Centralization:** Migrated all verified organization data into a structured Microsoft Teams Hub.
* **Governance:** Established role-based permissions and shared calendars for the "Fall 26' Launch" initiative.

## Tech Stack
* **Languages:** Python (Playwright, BeautifulSoup)
* **Platforms:** Microsoft Teams, Microsoft 365, PawLink
* **Data Management:** Microsoft Excel (Cleaned data exports)
